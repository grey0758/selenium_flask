# Selenium Web Automation with Flask

## 项目描述

该项目使用Flask框架构建了一个Web API，以便与Selenium进行交互，实现浏览器自动化操作。主要功能包括启动和停止Selenium、打开指定URL、与网页元素交互、保存和加载Cookies、以及使用BrowserMob Proxy拦截和修改网络请求。

## 安装与配置

### 环境要求

- Python 3.x
- Google Chrome 浏览器
- ChromeDriver
- Java 8 环境 (BrowserMob Proxy 需要)
